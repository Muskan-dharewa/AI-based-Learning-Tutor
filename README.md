# AI-based-Learning-Tutor
An AI-powered learning application that helps students 
understand any topic in an easy and interactive way. The system allows the user to enter a 
topic and select a difficulty level (Beginner, Intermediate, or Advanced). Based on the input, 
the AI generates a clear explanation of the topic, multiple-choice questions (MCQs), practice 
questions and detailed solutions. This helps students learn concepts step by step and test 
their understanding immediately.
